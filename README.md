# Paradise Nursery Shopping Application

A React-based e-commerce application for purchasing houseplants.

## Features
- **Landing Page:** Welcome screen with company information.
- **Product Listing:** Browse plants by category (Air Purifying, Aromatic, Low Maintenance).
- **Redux State Management:** Add plants to the cart and see quantities update live.
- **Shopping Cart:** Adjust quantities, remove items, and see total costs.

## Tech Stack
- React
- Redux Toolkit
- CSS3
